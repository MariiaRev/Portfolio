IVMDefiningSystem - an intelligent verse meter defining system based on a production rules model.
(Інтелектуальна система визначення віршового розміру з використанням продукційної моделі представлення знань).

Програма призначена для визначення віршового розміру українських віршів.

Дане програмне забезпечення дозволяється використовувати у некомерційних цілях із зазначенням авторства.

На даному етапі представлено демонстративний варіант програми (програма працює для віршів з папки Test, пізніше буде доданий повний словник, і програма працюватиме для будь-яких віршів).

Вкладка Інфо та її підвкладки знаходяться на опрацюванні.
